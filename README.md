# Redesign of Guruvayanakere Junction: Roundabout Framework
### Civil Engineering Project | Bantwal, Mangaluru

## 📌 Project Overview
This project focuses on the redesign of the **Guruvayanakere Junction** in Bantwal. This intersection is a critical bottleneck where State Highway 118 meets the Karkala-Belthangady road. 

The goal is to replace the current layout with a **modern roundabout (rotary)** designed according to **IRC:65-2017** standards to improve safety and traffic flow for pilgrims and local commuters.

## 🚦 Key Challenges
* **High Traffic Volume:** Major convergence point for traffic from Moodabidri, Karkala, and Puttur.
* **Safety Concerns:** Identifying conflict points in the current junction geometry.
* **Recent Developments:** Aligning the design with the ₹13 crore SH 118 upgrade project.

## 🏗️ Phase 1 Goals (Current Status)

* [ ] **Rotary Design:** Calculating entry/exit radii and weaving capacity per IRC guidelines.
